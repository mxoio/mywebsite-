Privacy Policy for mxioi.uk

Effective Date: April 6, 2025

This Privacy Policy describes how mxioi.uk ("we", "us", or "our") handles information in connection with the operation of this website, which is hosted on GitHub Pages and utilizes services from Cloudflare.

This website is primarily a static informational site displaying a countdown and related content. We aim to collect minimal personal data.

Information We Process via Third Parties

We rely on third-party services to host and operate this website. These services may collect certain information as part of their standard operations:

  • GitHub Pages (Hosting): This website is hosted on GitHub Pages. GitHub may collect standard server log information, such as your IP address, browser type, operating system, referring URLs, and timestamps when you access the site. This information is typically used for security monitoring, troubleshooting, and analyzing site usage trends. For more details, please refer to the GitHub Privacy Statement.
  • Cloudflare (DNS & Security): We use Cloudflare for DNS resolution and potentially for security services like Turnstile.
    • Cloudflare processes DNS queries made when your browser looks up our domain name.
    • If Cloudflare Turnstile is active, it collects minimal, non-intrusive data necessary to distinguish between human users and automated bots, primarily for security purposes. According to Cloudflare, Turnstile avoids using tracking cookies or harvesting sensitive device data.
    • Cloudflare may collect log data related to traffic routing and security events.
    For more details on Cloudflare's data practices, please see the Cloudflare Privacy Policy and information about Turnstile.

Cookies

This website itself does not set any non-essential cookies. However, our service providers (GitHub or Cloudflare) might use essential cookies required for security, network management, or basic functionality. For example, Cloudflare may place a strictly necessary cookie for managing security rules or Turnstile functionality.

How We Use Information

Any information collected by our third-party providers (GitHub, Cloudflare) is primarily used by them for:

  • Operating, maintaining, and securing the hosting and network infrastructure.
  • Providing DNS resolution and security services (like bot detection via Turnstile).
  • Monitoring website traffic for performance and security analysis (usually in aggregated or anonymized forms).

We do not directly access or process raw personal data collected by these providers beyond potentially viewing aggregated statistics.

Data Sharing

We do not sell or rent personal information. Data collected by GitHub and Cloudflare is subject to their respective privacy policies regarding sharing.

Data Security

We rely on the security measures implemented by GitHub and Cloudflare to protect the infrastructure hosting this website and the data they process.
